# Admin experience override

- Mood: calm operations center; clear, compact and predictable.
- Surface: light glass-tinted panels on a pale forest background; dark forest navigation.
- Accent: green for primary actions, gold for brand/context, amber/rose only for operational status.
- Navigation: persistent sidebar on desktop; modal drawer with backdrop and Escape support below 900px.
- Data: tables become labeled cards on narrow screens; action controls remain visible and touch-friendly.
- Forms: visible labels, inline errors, autofocus/focus trap in modals, minimum 44px interactive targets.
